# TODO API Documentation

## üìå Overview
This is a RESTful API for managing TODO tasks using **Node.js, Express, and MongoDB**. The API supports:

- Creating tasks
- Fetching all tasks
- Fetching tasks in progress
- Fetching completed tasks

## üöÄ Base URL
```
http://localhost:5000
```

---

## üìå Endpoints

### **1Ô∏è‚É£ Create a Task**
**Endpoint:** `POST /tasks`

**Request Body (JSON):**
```json
{
  "title": "Buy Groceries",
  "description": "Milk, Bread, and Eggs",
  "status": "pending",  
  "priority": "high",
  "dueDate": "2025-03-10T00:00:00Z"

  "description": "Milk, Bread, and Eggs",
  "status": "pending",
  "priority": "high",
  "dueDate": "2025-03-10T00:00:00Z",
  "createdAt": "2025-03-04T12:34:56Z",
  "__v": 0
}
```
